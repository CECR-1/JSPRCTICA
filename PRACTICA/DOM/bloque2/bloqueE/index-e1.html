Fetch API y REST - Conceptos Fundamentales

⚡ Sección 1: ¿Qué es Fetch API?

🚀 Fetch API

Fetch API es la herramienta nativa de JavaScript para hacer peticiones HTTP a servidores y APIs.

🎯 Puntos clave:

  • 📡 Función: Conecta tu JavaScript con APIs externas
  • 🌐 Nativo: Viene integrado en todos los navegadores modernos
  • 🔄 Asíncrono: No bloquea la interfaz mientras espera respuesta
  • 📊 Flexible: Maneja JSON, texto, archivos, etc.

💻 Sintaxis básica:

fetch('https://api.ejemplo.com/datos')
  .then(response => response.json())
  .then(data => console.log(data));

🏪 Analogía: Ir de compras

1 fetch(url) = Ir a la tienda
2 response = El vendedor te atiende
3 .json() = Abrir la bolsa de compras
4 data = Usar lo que compraste

🌐 Sección 2: ¿Qué es API REST?

🏗️ API REST

REST (REpresentational State Transfer) son las reglas para organizar y estructurar APIs de forma estándar y predecible.

🎯 Puntos clave:

  • 📋 Reglas: Cómo organizar endpoints y URLs
  • 🔄 Métodos HTTP: GET, POST, PUT, DELETE para CRUD
  • 🗂️ Recursos: URLs que representan datos (/users, /posts)
  • 📊 Sin estado: Cada petición es independiente

🌳 Estructura REST típica:

GET /api/usuarios → Lista todos
GET /api/usuarios/123 → Usuario específico
POST /api/usuarios → Crear nuevo
PUT /api/usuarios/123 → Actualizar
DELETE /api/usuarios/123 → Eliminar

🏢 Analogía: Sistema de oficinas

🏢 /usuarios = Departamento de RRHH
👤 /usuarios/123 = Escritorio del empleado 123
📋 GET = Consultar información
✏️ POST/PUT = Crear/actualizar expediente

🔄 Sección 3: Métodos HTTP - El lenguaje de las APIs

GET 📥

OBTENER

¿Qué hace? Solicita información del servidor

¿Modifica datos? No, solo lee

Ejemplo: Ver lista de productos

GET /api/productos

POST 📤

CREAR

¿Qué hace? Envía datos para crear algo nuevo

¿Modifica datos? Sí, crea nuevos registros

Ejemplo: Registrar nuevo usuario

POST /api/usuarios

PUT ✏️

ACTUALIZAR

¿Qué hace? Actualiza un recurso completo

¿Modifica datos? Sí, reemplaza información

Ejemplo: Editar perfil completo

PUT /api/usuarios/123

DELETE 🗑️

ELIMINAR

¿Qué hace? Elimina un recurso específico

¿Modifica datos? Sí, borra información

Ejemplo: Eliminar producto

DELETE /api/productos/456
