¿Qué es JSON? Sintaxis y Estructura

📋 Sección 1: ¿Qué es JSON?

JSON = JavaScript Object Notation

  • 📄 Formato de texto para intercambiar datos
  • 🌐 Estándar universal usado en APIs
  • 💡 Fácil de leer para humanos y máquinas
  • 🔄 Independiente del lenguaje de programación

Ejemplo de JSON:

{
  "nombre": "Ana García",
  "edad": 28,
  "activo": true,
  "skills": ["JavaScript", "CSS", "HTML"],
  "direccion": {
    "ciudad": "Madrid",
    "pais": "España"
  }
}

🔍 Características de JSON:

  • Strings: Siempre entre comillas dobles "texto"
  • Numbers: Enteros o decimales sin comillas
  • Booleans: true o false (sin comillas)
  • Arrays: Lista de valores entre [ ]
  • Objects: Pares clave-valor entre { }
  • null: Representa un valor nulo

⚙️ Sección 2: Sintaxis Correcta

Comparación: JavaScript vs JSON

💻 JavaScript Object

{
  nombre: 'Juan',
  edad: 25,
  activo: true,
  saludar: function() {
    return 'Hola!';
  }
}

📄 JSON

{
  "nombre": "Juan",
  "edad": 25,
  "activo": true
}

🚨 Diferencias importantes:

  • JSON usa comillas dobles para las claves
  • JSON NO permite funciones
  • JSON NO permite comentarios
  • JSON NO permite undefined

  • Arrays: Para listas de elementos similares
