🕐 FUNDAMENTOS DEL DOM

📖 Conceptos Clave

¿Qué es el DOM?

DOM (Document Object Model) es una representación en forma de árbol de la estructura HTML de una página web.

  • Cada elemento HTML es un nodo
  • JavaScript puede acceder y modificar estos nodos
  • Los cambios se reflejan inmediatamente en la página

Árbol de Nodos


document (raíz)
├── html
    ├── head
    │   ├── title
    │   └── meta
    └── body
        ├── h1
        ├── p
        └── div
            ├── span
            └── button

innerHTML vs textContent

  • innerHTML: Interpreta HTML tags
  • textContent: Solo texto plano
  • value: Para inputs y formularios

Modificar Estilos CSS

  • element.style.property: Estilos inline
  • element.classList: Manipular clases CSS
  • add(), remove(), toggle(): Métodos de classList

¿Qué son los Eventos?

Los eventos son acciones que ocurren en el navegador:

  • click: Usuario hace clic
  • input: Usuario escribe en un campo
  • change: Valor de un elemento cambia
  • submit: Formulario se envía

addEventListener()


element.addEventListener('evento', function() {
    // Código a ejecutar
});
