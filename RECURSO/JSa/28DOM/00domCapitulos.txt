# Referencias para el Curso de DOM

## Libros

1. "DOM Enlightenment" por Cody Lindley
   - Cubre los fundamentos del DOM de manera profunda y práctica.

2. "JavaScript: The Definitive Guide" por David Flanagan
   - Incluye secciones extensas sobre el DOM y su manipulación.

3. "Eloquent JavaScript" por Marijn Haverbeke
   - Tiene capítulos dedicados al DOM y eventos del navegador.

4. "JavaScript & jQuery: The Missing Manual" por David Sawyer McFarland
   - Ofrece una perspectiva práctica sobre la manipulación del DOM.

5. "Professional JavaScript for Web Developers" por Matt Frisbie
   - Contiene capítulos detallados sobre el DOM y APIs relacionadas.

## Recursos en línea

1. MDN Web Docs - DOM
   https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model
   - Documentación completa y actualizada sobre el DOM.

2. W3Schools - JavaScript HTML DOM
   https://www.w3schools.com/js/js_htmldom.asp
   - Tutoriales y ejemplos prácticos sobre manipulación del DOM.

3. JavaScript.info - Document
   https://javascript.info/document
   - Guía detallada sobre el DOM con ejemplos interactivos.

4. CSS-Tricks - What Is the DOM?
   https://css-tricks.com/dom/
   - Artículo que explica el DOM de manera clara y concisa.

5. Udacity - JavaScript and the DOM (curso gratuito)
   https://www.udacity.com/course/javascript-and-the-dom--ud117
   - Curso en línea que cubre los fundamentos del DOM.

6. FreeCodeCamp - Manipulating the DOM
   https://www.freecodecamp.org/news/tag/dom/
   - Colección de artículos y tutoriales sobre manipulación del DOM.

7. Google Developers - Web Fundamentals: Working with the DOM
   https://developers.google.com/web/fundamentals/web-components/shadowdom
   - Guía sobre conceptos avanzados como Shadow DOM.

8. JavaScript30 por Wes Bos
   https://javascript30.com/
   - Curso gratuito de 30 días con varios proyectos que involucran manipulación del DOM.

9. DOM Manipulation in JavaScript - All You Need to Know
   https://www.freecodecamp.org/news/dom-manipulation-in-javascript/
   - Guía completa de FreeCodeCamp sobre manipulación del DOM.

10. You Don't Need jQuery
    http://youmightnotneedjquery.com/
    - Muestra cómo realizar tareas comunes del DOM con JavaScript puro.
