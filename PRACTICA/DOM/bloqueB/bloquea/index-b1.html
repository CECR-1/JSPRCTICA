📝 Sección 1: innerHTML vs textContent

Contenido original con HTML

🔍 Diferencias clave:

  • innerHTML: Interpreta HTML → etiquetas se convierten en elementos
  • textContent: Solo texto plano → etiquetas se muestran como texto
  • Uso: innerHTML para contenido HTML, textContent para texto seguro
